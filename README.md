CS 467 Auto Parts Store - Group 6A

auto parts store for the cs 467 capstone. lets you browse parts, add to cart, checkout with a real credit card. has an admin panel, warehouse pack/ship flow, and a receiving page for inventory.

stack:
- node + express
- plain html/css/js
- mysql on railway for orders/inventory
- NIU parts database (read only)
- NIU credit card processor
- emails through gmail/nodemailer
- hosted on render

to run:
npm install
node server.js
localhost:3000

db connects to railway by default, change localdb.js if needed

admin/warehouse login: admin1 / aps123
